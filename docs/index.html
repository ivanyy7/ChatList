О программе

ChatList — приложение для отправки одного промпта в несколько нейросетей и сравнения их ответов. Введите запрос, выберите модели и получите ответы в удобной таблице.

Возможности

  • Отправка промпта в несколько моделей одновременно
  • Сохранение промптов и результатов в базу данных
  • Экспорт в Markdown и JSON
  • Управление моделями через удобный интерфейс
  • AI-улучшение промптов
  • Логирование всех запросов и действий

Поддерживаемые API

OpenAI, DeepSeek, Groq, OpenRouter, HuggingFace и другие совместимые с OpenAI API.
